Version14 変更内容:
- 日付ごとのスケジュール表示を廃止
- 活動スケジュールを3カード形式に変更
  1. 火曜日・水曜日：女子メイン／青山・瀬田周辺
  2. 木曜日・金曜日：男子メイン／膳所・皇子山周辺
  3. 土曜日・日曜日・祝日：県内外遠征／昼は瀬田・夜は唐崎
- 管理画面のスケジュールタブから曜日、対象、場所、時間帯、説明、アイコン、色を編集可能
- 公開サイトのスケジュール部分を「行きたい」と感じられるカードデザインへ変更
- 下部バージョン表示を 14.0.0 に更新
